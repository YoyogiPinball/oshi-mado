# 推し窓（oshi-mado）

推し（YouTube チャンネル）の新着を横断でチェックする Chrome 拡張機能。ツールバーのアイコンを押すと、登録した推しの最近の投稿が小さな窓（ポップアップ）に集まります。

**全部 / 動画 / 配信中 / アーカイブ / 予定** のタブと、チャンネル・日付・時刻のフィルタで1画面に見渡せます。時刻は「本体が視聴可能になったタイミング」（動画=公開時刻、配信=開始時刻、予定=開始予定時刻）を基準にします。

## 構成

- **Chrome 拡張機能（Manifest V3）** — `extension/` 配下が本体
  - `popup.html` / `popup.js` — 新着ダイジェスト（ポップアップ、`?view=tab` で通常タブ表示）
  - `options.html` / `options.js` — 推しリストの編集・API キー・「動画をアプリで開く」設定
- **データ源は2つ**
  - 新着の発見 = 各チャンネルの公開 RSS（クォータ消費なし）
  - 種類判定と配信時刻 = YouTube Data API v3 `videos.list`（キー任意。無くても新着一覧は出る）
- **保存先** = `chrome.storage.sync`（推しリスト・API キー・設定）、`chrome.storage.local`（判定キャッシュ）

## 使い方

1. `chrome://extensions` で「デベロッパーモード」を ON にし、「パッケージ化されていない拡張機能を読み込む」で `extension/` を選ぶ。
2. ツールバーの 推し窓 アイコン → ⚙ から推しを登録（チャンネル ID / URL / @ハンドル）。
3. 配信の判定を使いたいときは、⚙ で YouTube Data API v3 のキーを設定（任意）。

## テスト

種類判定・尺パースなど API を叩かない純粋ロジックを対象に、静的チェック（構文＋未定義変数スキャン）で確認します。ブラウザ実機の挙動（`chrome.storage`・API 通信）は手動確認です。
