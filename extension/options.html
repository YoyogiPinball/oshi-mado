推しリスト

いま登録している推しの一覧です。アイコンと名前は APIキーがあれば自動で出ます（キーが無いときは名前だけ）。
「外す」のは「保存」を押すまで確定しません。「保存」を押す前なら「戻す」で元に戻せます。

推しを追加

チャンネルID（UC…） / チャンネルURL / @ハンドル のいずれでもOK。 複数まとめて足すときは1行に1つ貼り付けてください（一括インポートの受け口も兼ねます）。

YouTube Data API キー（任意）

配信の開始時刻や配信中／予定／アーカイブの判定、推しリストのアイコンと名前の取得に使います。 未設定でも新着一覧は出ますが、配信は「枠を立てた時刻」に並び、種類の判定はされません。

取得手順（初回のみ・数分で終わります）:

  1. Google Cloud Console を開き、上部でプロジェクトを1つ作成します（既存のものでも構いません）。
  2. YouTube Data API v3 のページで「有効にする」を押します。
  3. 「APIとサービス」→「認証情報」→「認証情報を作成」→「APIキー」を選び、表示された AIza… をコピーします。
  4. 作成したキーを開き、「アプリケーションの制限」は「なし」にします。拡張機能にはリファラー制限が効かないため、制限をかけると通信が弾かれます。
  5. （推奨）「APIの制限」で YouTube Data API v3 のみに絞ると、万一漏れても他のAPIには使えません。
  6. 下の欄に貼り付けて「保存して接続テスト」を押します。

キーの扱いについて
・このキーは chrome.storage.sync に保存され、あなたがログイン中の Chrome 全体に同期されます（他人とは共有されません）。
・送信先は Google の API のみです。推し窓が外部のサーバーへ送ることはありません。
・他人に渡したり、公開リポジトリ・スクショに貼ったりしないでください。
・無料枠は1日あたり1万ユニット。推し窓の消費はごく軽微で、通常は使い切りません（超えても翌日に戻ります）。
・漏れたかも、と思ったら Cloud Console でそのキーを削除し、作り直して入れ直してください。

動画を YouTube アプリで開くには

カードは通常、ブラウザの新しいタブで開きます。デスクトップの YouTube アプリで開きたい場合は、 Chrome 側を設定します（推し窓側の操作は不要です）。

  1. Chrome で YouTube を開き、アドレスバー右の「インストール」アイコン、または「⋮」→「キャストして保存して共有」→「ページをアプリとしてインストール」で YouTube をアプリ化します。
  2. chrome://apps を開き、YouTube アプリを右クリック →「対応リンクをこのアプリで開く」を有効にします。

※ 環境によっては、拡張機能のポップアップから開くリンクがリンクキャプチャの対象外となり、 上記を設定してもブラウザのタブで開くことがあります。その場合はブラウザ表示のままご利用ください。
