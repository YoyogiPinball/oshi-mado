推し窓 設定

動画の開き方

これを ON にすると、カードをクリックしたとき opener を持たない新規タブとして開き、 Chrome のリンクキャプチャでアプリに渡せるようになります。
ただし Chrome 側の準備も必要です。YouTube をアプリとしてインストールし、 そのアプリ設定で「対応リンクをこのアプリで開く」を有効にしてください。 どちらか一方だけではブラウザのタブで開きます。
Ctrl／⌘／中クリックで開いたときは、この設定に関わらず従来どおりタブで開きます。

YouTube Data API キー

配信の開始時刻や配信中／予定の判定に使います。未設定でも新着一覧は出ますが、 配信は「枠を立てた時刻」に並び、種類の判定もされません。
Google Cloud Console で YouTube Data API v3 を有効化 →「認証情報」→「APIキー」で取得できます。

推しリスト

いま登録している推しの一覧です。アイコンと名前は APIキーがあれば自動で出ます（キーが無いときは名前だけ）。
「外す」のは「保存」を押すまで確定しません。押す前なら「戻す」で元に戻せます。
